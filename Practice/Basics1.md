## Задание 14 — Реализовать пузырьковую сортировку для массива фиксированного размера

**Цель:** написать функцию `bubbleSort`, которая сортирует фиксированный массив целых чисел *вместе с примером использования в `main`*.

---

### Условие

Реализуйте функцию

```go
func bubbleSort(a *[5]int)
```

которая выполняет сортировку массива `a` длины **5** методом пузырька (по возрастанию). Функция должна сортировать массив **in-place** (то есть менять элементы самого массива, переданного по указателю) и не возвращать значение.

В `main`:

1. Считайте 5 целых чисел из стандартного ввода.
2. Вызовите `bubbleSort`.
3. Выведите элементы отсортированного массива в одну строку через пробел.

> Почему `*[5]int`? Потому что массив фиксированного размера при передаче в функцию копируется — передав указатель, вы сортируете оригинал и избегаете лишнего копирования.

---

### Формат ввода

Одна строка или несколько — 5 целых чисел (через пробел/новую строку):

```
a1 a2 a3 a4 a5
```

### Формат вывода

Одна строка — 5 чисел в порядке возрастания, разделённых одним пробелом:

```
v1 v2 v3 v4 v5
```

---

### Пример

**Ввод**

```
3 1 4 1 5
```

**Вывод**

```
1 1 3 4 5
```

---

### Подсказки

* Поменять элементы местами можно так:

  ```go
  a[j], a[j+1] = a[j+1], a[j]
  ```
* Для `n=5` используйте константу или `len(a)` (через `len(a)` для указателя: `len(a)` не работает; можно писать `n := len(a)` *до* распаковки — в данном конкретном случае проще использовать `const n = 5` или работать с срезом `a[:]`).
* Если за проход не произошло ни одной перестановки — массив уже отсортирован, можно досрочно завершить сортировку.
* Сложность: O(n²) в худшем/среднем случае.

---

### Шаблон 

```go
package main

import (
    "fmt"
)

func bubbleSort(a *[5]int) {

}

func main() {
    var arr [5]int
    // читает 5 чисел из stdin (можно разделять пробелами или новыми строками)
    for i := 0; i < 5; i++ {
        if _, err := fmt.Scan(&arr[i]); err != nil {
            return
        }
    }

    bubbleSort(&arr)

    for i, v := range arr {
        if i > 0 {
            fmt.Print(" ")
        }
        fmt.Print(v)
    }
    fmt.Println()
}
```

---

### Проверка

* Ввод уже отсортированного массива: `1 2 3 4 5` → выход тот же.
* Обратный порядок: `5 4 3 2 1` → `1 2 3 4 5`.
* Повторяющиеся элементы: `2 2 2 2 2` → `2 2 2 2 2`.
* Отрицательные и положительные: `0 -1 5 -3 2`.
